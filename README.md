# Lmart 포트폴리오

## 소개
실제 대형마트 웹사이트를 참고하여, UI/UX 구조를 재구성하고 사용자 친화적인 화면을 구현한 프로젝트입니다.

## 사용 기술
- HTML
- CSS
- JavaScript
- jQuery

## 주요 기능
- 상품 슬라이더
- 탭 인터페이스
- 3단계 동적 메뉴 구현 (fetch + split)
- 반응형 레이아웃 적용
